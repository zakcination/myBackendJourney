Welcome to Project Movie

This project is a simple movie rating system.

Movies

This page displays a list of all movies in our database. For each movie, you can see its title and ID.

Ratings

This page displays a list of all movies along with their ratings. For each movie, you can see its title, ID, the number of reviews, the average rating, and each individual review.

Reviewers

This page displays a list of all reviewers. For each reviewer, you can see their name and the reviews they've written.
